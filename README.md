# Proyecto de Automatización Urban Routes

## Descripción
Este proyecto contiene una suite de pruebas automatizadas utilizando **Selenium WebDriver** y **Python**. El objetivo es validar el flujo completo de reserva de un taxi en la plataforma Urban Routes, desde la selección de direcciones hasta la asignación de un conductor.

## Tecnologías y Técnicas
- **Lenguaje:** Python 3.x
- **Framework de Pruebas:** Pytest
- **Automatización:** Selenium WebDriver
- **Patrón de Diseño:** Page Object Model (POM) para mejorar la mantenibilidad.
- **Intercepción de Datos:** Uso de funciones auxiliares para capturar códigos de confirmación SMS.
